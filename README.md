# How to install

```
cd ~/.conky
git clone https://github.com/DavideAlwaysMe/conky-dashboard.git
conky -c ~/.conky/conky-dashboard/.conkyrc
```
## Dependencies

- conky-all
- [caviar dreams](https://www.dafont.com/caviar-dreams.font) font
